### 是否可变

- 可变
	- 列表
	- 集合
	- 字典
- 不可变
	- 元组
	- 字符串
	- 字节序列
	- 不可变集合
- 最后我们来个表格


### 表格

|  类型名称   | 描述  |  是否可变类型   |是否有顺序 | 举例 |
|  ----  | ----   | ----  | ----  |----  |
| 列表 | list| 是 | 是 | [1,2,3]、["a","b"],list(1,2)|
| 元组  | tuple | 否  |是 | (1,2) |
| 字符串  | str|否 | 是 | "oeasy"  |
| 字节序列  | bytes |否 |是 |  b"oeasy"  |
| 集合  | set | 是 |否  | set([1,2,3]) |
| 不可变集合 | frozeset | 否  |否 |  frozenset([1,2,3]) |
| 字典 | dict | 是 |否  |  {'a':1,'b':2} |
