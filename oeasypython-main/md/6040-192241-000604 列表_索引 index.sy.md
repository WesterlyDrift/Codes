# 列表 - 构造
## 回忆

- 这次了解了列表生成
- 列表作为一个序列容器
- 可以通过其他序列转化得到
- 数值序列range
	- start、stop、step
	- 负数
	- 前闭后开
- 字符串序列str
	- 把每个字符转化为列表项
	- split切分出列表项
	- 中文无压力
- 字节流序列byte
	- 可以直接把字节流转化为列表
	- 每个列表项都是一个字节
- 列表还能做什么呢？🤔
