# 集合 -集合概念
## 回忆
- 上次复习了字节序列
- 字节序列也是序列
- 和字符串和列表、元组一样都属于序列
- 字节可以喝16进制字符串相互转化
- 字节序列可以做序列的一切操作
	- 索引
	- 切片
	- 运算

### 冰封集合
- frozenset
- 构造之后就不能增删改了
- 就像元组一样
- 列表可以改，单元组不行
- 集合可以改，但冰封集合不行
